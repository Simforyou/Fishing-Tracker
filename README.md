
# Fishing Tracker Home Assistant Integration

Version 0.3.0

## Enthalten
- Beißchance Sensor
- Beste Angelzeit
- KI Empfehlung
- Fanghistorie
- Wassertemperatur
- Fang speichern / Kein Fang
- Fischart / Spot / Köder Auswahl
- HACS kompatibel

## Installation
Über HACS als benutzerdefiniertes Repository installieren.
