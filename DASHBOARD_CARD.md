# Fishing Tracker Auto-Dashboard Karte

Falls das Sidebar-Panel in deiner Home-Assistant-Version nicht unterstützt wird,
füge diese Karte in ein normales Dashboard ein:

```yaml
type: iframe
url: /local/fishing_tracker_dashboard.html
aspect_ratio: 190%
```

Alternativ direkt im Browser öffnen:

`/local/fishing_tracker_dashboard.html`
